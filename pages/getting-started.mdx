# üêæ Getting Started with Anima Unity Microservices

Welcome to **Anima Unity** - Indonesia's first comprehensive microservices-based digital ecosystem for animal welfare, rescue, and empowerment. This developer guide will help you set up your microservices development environment and start building with our distributed platform.

<Callout type="info">
**"Tidak ada hewan yang dibiarkan tertinggal"** - No animal left behind. Our mission is to provide digital identity, access, protection, and care for every pet and wild animal through scalable microservices architecture, community collaboration, and compassionate technology.
</Callout>

<Steps>
### Prerequisites Installation

Ensure you have the following tools installed:

```bash
# Container orchestration
docker --version          # >= 24.0
docker-compose --version  # >= 2.20

# Kubernetes (optional for local development)
kubectl version --client  # >= 1.28
minikube version          # >= 1.31

# Service mesh (optional)
istioctl version          # >= 1.19

# Development tools
git --version             # >= 2.40
node --version            # >= 18.0
java --version            # >= 17
python --version          # >= 3.9
go version                # >= 1.21
cargo --version           # >= 1.70
```

```bash
git clone https://github.com/anima-unity/microservices-platform.git
cd anima-unity-microservices
```

    export async function reportRescueCase(caseData: RescueCase) {
      const span = tracer.startSpan('report_rescue_case');
      
      try {

          'rescue.urgency': caseData.urgency,
          'rescue.location': JSON.stringify(caseData.location)

        span.setStatus({ code: trace.SpanStatusCode.OK });
        return result;

        span.setStatus({

        });
        throw error;
      } finally {
        span.end();
      }
    }
